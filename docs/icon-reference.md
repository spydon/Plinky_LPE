# Icon Reference

All icons available for use in documentation. Reference any icon with `![icon_name]` — definitions
are in `docs/includes/icons.md`.

To control size, use the `attr_list` syntax:  
`![delay]{ .icon-lg }`  
or inline style:  
`![delay]{ style="width:48px;height:48px" }`.

The default size (no class) is **32px**. Available size classes:

| Class      | Size | Example             |
| ---------- | ---- | ------------------- |
| *(none)*   | 32px | ![play]             |
| `.icon-sm` | 16px | ![play]{ .icon-sm } |
| `.icon-md` | 24px | ![play]{ .icon-md } |
| `.icon-lg` | 32px | ![play]{ .icon-lg } |
| `.icon-xl` | 48px | ![play]{ .icon-xl } |
