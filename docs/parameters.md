# Parameters

Parameters are arranged in a grid of rows and columns. The **Param 1** and **Param 2** shift buttons select two layers per row. Each row corresponds to a functional area; each column corresponds to a pad position (1–6).

## ![ab]{ .icon-sm } LFOs (A, B, X, Y)

All four LFOs share the same parameter layout. The **AB** row selects LFO A (Param 1) and LFO B (Param 2); the **XY** row selects LFO X (Param 1) and LFO Y (Param 2).
